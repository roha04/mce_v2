# MCE v2 — Метод скінченних елементів для аналізу тривимірних задач теорії пружності

## 📋 Опис програми

**MCE v2** — це програма для числового аналізу напружено-деформованого стану (НДС) тривимірного пружного тіла на основі **методу скінченних елементів (МСЕ)**. Програма реалізує повний цикл розрахунку: від генерації сітки до візуалізації результатів.

### Фізична модель
Розглядається **прямокутний паралелепіпед** розмірами **a × b × c**:
- **Нижня грань** (z = 0) — жорстко закріплена (переміщення = 0)
- **Верхня грань** (z = c) — прикладене нормальне навантаження (тиск **P**)
- Матеріал — **ізотропний пружний** (характеризується модулем Юнга **E** та коефіцієнтом Пуассона **ν**)

### Дискретизація
Використовуються **20-вузлові серендипові скінченні елементи** (криволінійні шестигранники). Кожен елемент має:
- **8 кутових вузлів** (вершини)
- **12 серединних вузлів** (на серединах ребер)

## 🚀 Як запустити

### Вимоги
- **Windows 10/11**
- **Visual Studio** (2019 або новіший) або **MSBuild**
- **CMake** 3.14+
- **Git** (для завантаження залежностей)

### Збірка та запуск

```bash
# Клонувати репозиторій
git clone https://github.com/roha04/mce_v2.git
cd mce_v2

# Налаштувати проект (CMake автоматично завантажить Eigen, GLFW, ImGui)
cmake -S . -B build

# Скомпілювати
cmake --build build

# Запустити
.\build\Debug\MCE_App.exe
```

Залежності завантажуються автоматично через `FetchContent`:
- **Eigen 3.4** — лінійна алгебра (матриці, вектори)
- **GLFW 3.3.8** — створення вікна OpenGL
- **ImGui 1.91** — графічний інтерфейс користувача

## ⚙️ Параметри задачі (вкладка "Введення даних")

### Геометрія паралелепіпеда

| Параметр | Опис | Типове значення |
|----------|------|-----------------|
| **Довжина a (X)** | Розмір уздовж осі X | 10.0 |
| **Ширина b (Y)** | Розмір уздовж осі Y | 5.0 |
| **Висота c (Z)** | Розмір уздовж осі Z | 2.0 |

### Параметри сітки

| Параметр | Опис | Типове значення |
|----------|------|-----------------|
| **nx** | Кількість елементів уздовж X | 2 |
| **ny** | Кількість елементів уздовж Y | 2 |
| **nz** | Кількість елементів уздовж Z | 1 |

> **Загальна кількість вузлів**: nqp = (2·nx + 1) × (2·ny + 1) × (2·nz + 1)
> **Загальна кількість елементів**: nel = nx × ny × nz

### Характеристики матеріалу

| Параметр | Опис | Типове значення |
|----------|------|-----------------|
| **E** | Модуль Юнга (Па) | 2.1×10¹¹ (сталь) |
| **ν (nu)** | Коефіцієнт Пуассона | 0.3 |

**Коефіцієнти Ляме** обчислюються автоматично:
```
λ = E·ν / ((1+ν)(1-2ν))
μ = E / (2(1+ν))        — модуль зсуву
```

### Навантаження

| Параметр | Опис | Типове значення |
|----------|------|-----------------|
| **P** | Тиск на верхню грань (Па) | 100.0 |

> Навантаження прикладене **нормально** (перпендикулярно) до верхньої грані z = c.

### Кнопки

| Кнопка | Дія |
|--------|-----|
| **Згенерувати сітку** | Створює сітку 20-вузлових елементів |
| **Повний розрахунок МСЕ** | Виконує всі кроки: базисні функції → збірка → розв'язування → напруження |

## 🔬 Математична модель

### Рівняння теорії пружності

Програма розв'язує систему рівнянь статичної теорії пружності:

**1. Рівняння рівноваги:**
```
∂σ_xx/∂x + ∂σ_xy/∂y + ∂σ_xz/∂z = 0
∂σ_xy/∂x + ∂σ_yy/∂y + ∂σ_yz/∂z = 0
∂σ_xz/∂x + ∂σ_yz/∂y + ∂σ_zz/∂z = 0
```

**2. Закон Гука (ізотропне тіло):**
```
σ_xx = λ·θ + 2μ·ε_xx
σ_yy = λ·θ + 2μ·ε_yy
σ_zz = λ·θ + 2μ·ε_zz
σ_xy = μ·γ_xy
σ_yz = μ·γ_yz
σ_xz = μ·γ_xz
```
де θ = ε_xx + ε_yy + ε_zz — об'ємна деформація

**3. Деформації через переміщення:**
```
ε_xx = ∂u/∂x,  ε_yy = ∂v/∂y,  ε_zz = ∂w/∂z
γ_xy = ∂u/∂y + ∂v/∂x
γ_yz = ∂v/∂z + ∂w/∂y
γ_xz = ∂u/∂z + ∂w/∂x
```

### Варіаційне формулювання

Задача зводиться до варіаційної:
```
a(u, v) = l(v),  ∀v ∈ V
```

### Метод скінченних елементів

Область ділиться на **nel** 20-вузлових елементів. На кожному елементі:
- Апроксимація переміщень: u = Σ N_i · u_i
- Матриця жорсткості: K_e[60×60]
- Вектор навантаження: F_e[60]
- Глобальна система: K·U = F

Чисельне інтегрування — **квадратура Гауса** (3×3×3 = 27 точок).

### Напруження (вкладка "Напруження")

**Компоненти тензора напружень:**
| Стовпець | Опис |
|----------|------|
| **σ_xx** | Нормальне напруження за X (Па) |
| **σ_yy** | Нормальне напруження за Y (Па) |
| **σ_zz** | Нормальне напруження за Z (Па) |
| **σ_xy** | Дотичне напруження в площині XY (Па) |
| **σ_yz** | Дотичне напруження в площині YZ (Па) |
| **σ_xz** | Дотичне напруження в площині XZ (Па) |

**Головні напруження:**
| Стовпець | Опис |
|----------|------|
| **σ₁** | Максимальне головне напруження (Па) |
| **σ₂** | Середнє головне напруження (Па) |
| **σ₃** | Мінімальне головне напруження (Па) |

**Напруження за Мізесом:**
```
σ_vM = √(0.5·[(σ₁-σ₂)² + (σ₂-σ₃)² + (σ₃-σ₁)²])
```

## 📐 Чисельні методи

### Квадратура Гауса
Інтегрування по «стандартному» кубу [-1,1]³:
```c++
∫∫∫ f(ξ,η,ζ) dξ dη dζ ≈ ΣᵢΣⱼΣₖ cᵢ·cⱼ·cₖ·f(ξᵢ, ηⱼ, ζₖ)
```
- **3 вузли** на напрямок → **27 точок** для об'ємних інтегралів
- **3×3 = 9 точок** для поверхневих інтегралів

Вузли Гауса: ξ = {-√(3/5), 0, √(3/5)}
Ваги: w = {5/9, 8/9, 5/9}

### Розв'язувач СЛАР
- **Метод Гауса** для стрічкової симетричної матриці
- Півширина стрічки: ng = 3·(N_max - N_min + 1) на елементі
- **Метод штрафу** для крайових умов (діагональ = 10³⁰)

### Матриця жорсткості
Формується з блоків a_ij (формули 15):
```
a₁₁ = (λ+2μ)·Nᵢ'ₓ·Nⱼ'ₓ + μ·(Nᵢ'ᵧ·Nⱼ'ᵧ + Nᵢ'₂·Nⱼ'₂)
a₁₂ = λ·Nᵢ'ₓ·Nⱼ'ᵧ + μ·Nᵢ'ᵧ·Nⱼ'ₓ
... і т.д.
```

## 💡 Поради

1. **Починайте з грубої сітки** (2×2×1) для швидкого тестування
2. **Збільшуйте nx, ny, nz** для точніших результатів (2×2×2 = 8 ел., 3×3×3 = 27 ел.)
3. **Масштаб деформації** — регулюйте клавішами ↑/↓, щоб побачити деформацію (реальні переміщення дуже малі)
4. **Фільтр** — використовуйте для пошуку конкретних вузлів у таблицях
5. **Матеріали**: сталь (E=2.1e11, ν=0.3), алюміній (E=7e10, ν=0.34), бетон (E=3e10, ν=0.2)
